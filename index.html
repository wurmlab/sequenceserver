SequenceServer Beta

Easily set up a local BLAST web server

to search & share sequence data.


Sequenceserver is an efficient and elegant alternative to NCBI's wwwblast: with Sequenceserver you can rapidly set up a local BLAST server with a helpful user interface.

Sequenceserver will eventually become "The iPod of Blast searching". This beta version is fully functional and has already deployed for several databases (eg: demo ant genome BLAST server).

Features

Easy setup in five minutes:

  • Helps you batch format databases if required.
  • Finds all formatted blast databases at start-up.
  • No configuration necessary: uses ruby's internal web servers. Or deploy on Apache/Nginix with Rack.

Easy to use:

  • Elegant & "idiot-proof" interface:
  • Deactivates inappropriate blast methods depending on type of entered sequence.
  • Deactivates inappropriate databases based on selected blast method.
  • Single-click retrieval of hit sequences directly from BLAST results (you can easily add custom links)

Demo server

Ant genome sequence blast search: running a pre-release version of SequenceServer. Hosted by the ant genome database.

Requirements

  • ruby 1.8.7 or later (if you have git, get ruby with RVM).
  • NCBI Blast+ (thoroughly tested with 2.2.25, some issues exist with earlier versions; can use system-wide blast installtion or a specific directory)
  • A folder containing FASTA sequence files

Download & Setup

Extensive details in the README file. In short:
  • Download sequenceserver. Either with this link or copy-paste the following into the command-line: 
    $ wget http://github.com/yannickwurm/sequenceserver/zipball/master -O sequenceserver.zip
  • Unzip, and move to Sequence Server's directory, and do: 
    $ gem install bundler && bundle install
  • Edit the example.config.yml file to indicate the location of your blast databases. Or create some with the tool we provide: database_formatter.rb directory_of_fasta_files; or manually with Blast's makeblastdb -in my.fasta -dbtype nucl -parse_seqids.
  • Launch the server: 
    $ ruby sequenceserver.rb
  • Access SequenceServer on http://localhost:4567

Sequenceserver is an alternative to NCBI's wwwblast, Viroblast, and GMOD

There are several. But they are suboptimal because they look like they're from the last millenium. And installing them is challenging.

  • NCBI's wwwblast hasn't been maintained in years and is difficult to set up.
  • ViroBLAST requires Apache & MySQL.
  • GMOD's BLAST Graphic Viewer hasn't been maintained in years.
  • BLASTstation is only for local BLAST.
